Thue

Thue is an esoteric programming language, which uses simple commands to iterate over a single string.

Note: this editor uses NOT the original language but Thue with syntax modifications and a few other modifications, being described below.

Main string is a single string over which the program iterates. In the editor, the main string is the last line of code.

One command is a line of code, consisting of an original string, operator -> (with spaces on the sides) and a replacement string. If the original string is found in the main string, it is replaced by the replacement string.

aaa -> xyz

To get a string from the input stream, set the replacement string to ~.

aaa -> ~

To print something to the console, start the replacement string with ~ and then type the string to be printed.

aaa -> ~Hello World!

Lines of code without -> operator are skipped, so you are free to make comments.

A very useful comment
a -> b
But the second line is not a comment

Note: don't use -> operator in original or replacement strings.
